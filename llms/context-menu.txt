# JamUI — Context Menu

> A fully accessible, multi-level context menu system with keyboard navigation, submenus, checkboxes, and radio groups for Svelte 5.

## Context Menu API & Usage Guide

The JamUI Context Menu is a composable set of Svelte 5 components that build a fully accessible right-click context menu. It uses Svelte 5 Runes (`$state`, `$derived`, `$effect`, `$props`) and context-based state management via `setContext`/`getContext` with Symbol keys. The architecture is split into a root provider, a trigger, a content container, and a rich set of child primitives (items, checkboxes, radio groups, separators, labels, submenus, shortcuts). It manages its own positioning with viewport boundary detection, animated open/close transitions, and full keyboard navigation following WAI-ARIA menu patterns.

### Component Structure

- `ContextMenu`: Root container that provides open state and position via context.
- `ContextMenuTrigger`: The element that triggers the context menu on right-click or keyboard.
- `ContextMenuContent`: The floating menu panel that positions itself and manages keyboard navigation.
- `ContextMenuItem`: A standard clickable menu item.
- `ContextMenuCheckboxItem`: A toggleable checkbox menu item with a check icon.
- `ContextMenuRadioGroup`: A container that manages a set of mutually exclusive radio items.
- `ContextMenuRadioItem`: A single radio option within a radio group.
- `ContextMenuSeparator`: A visual divider between menu items.
- `ContextMenuLabel`: A non-interactive text label for grouping items.
- `ContextMenuGroup`: A semantic grouping wrapper for menu items.
- `ContextMenuShortcut`: Displays keyboard shortcut hints aligned to the right.
- `ContextMenuSub`: Root container for a submenu, managing its open/close state.
- `ContextMenuSubTrigger`: The trigger item that opens a submenu.
- `ContextMenuSubContent`: The floating panel containing submenu items.

## Components API

### ContextMenu.svelte

Root container component. Renders nothing visible — it provides context to all child components.

#### Props
- `children`: `Snippet` (optional). The menu's child components (trigger, content, etc.).

---

### ContextMenuTrigger.svelte

The element that opens the context menu when right-clicked or activated via keyboard.

#### Props
- `children`: `Snippet` (optional). Content rendered inside the trigger element.
- `class`: `string` (default: `''`). Additional CSS classes.
- `...rest`: `HTMLAttributes<HTMLDivElement>` — all standard HTML div attributes are spread onto the element.

---

### ContextMenuContent.svelte

The floating menu panel. Automatically positions itself at the cursor location with viewport boundary clamping. Manages keyboard navigation for all child items.

#### Props
- `children`: `Snippet` (optional). Menu items, separators, labels, and other content.
- `class`: `string` (default: `''`). Additional CSS classes.
- `alignOffset`: `number` (default: `0`). Horizontal offset applied to the menu position.
- `sideOffset`: `number` (default: `0`). Vertical offset applied to the menu position.
- `...rest`: `HTMLAttributes<HTMLDivElement>` — all standard HTML div attributes are spread onto the element.

---

### ContextMenuItem.svelte

A standard clickable menu item. Closes the menu on click.

#### Props
- `children`: `Snippet` (optional). Content of the menu item.
- `class`: `string` (default: `''`). Additional CSS classes.
- `destructive`: `boolean` (default: `false`). When true, the item uses danger/red styling on hover and focus.
- `disabled`: `boolean` (default: `false`). Disables the item — not focusable, not clickable.
- `onclick`: `(e: MouseEvent) => void` (optional). Click handler. Menu closes automatically after click.
- `...rest`: `HTMLButtonAttributes` — all standard HTML button attributes are spread onto the element.

---

### ContextMenuCheckboxItem.svelte

A toggleable checkbox menu item. Shows a check icon when checked.

#### Props
- `children`: `Snippet` (optional). Content of the checkbox item.
- `checked`: `boolean` (default: `false`). Whether the checkbox is currently checked.
- `class`: `string` (default: `''`). Additional CSS classes.
- `disabled`: `boolean` (default: `false`). Disables the item.
- `oncheckedchange`: `(checked: boolean) => void` (optional). Callback fired when the checked state toggles. The menu closes automatically after the change.
- `...rest`: `HTMLButtonAttributes` — all standard HTML button attributes are spread onto the element.

---

### ContextMenuRadioGroup.svelte

A container that manages a set of mutually exclusive radio items. Provides the current value and an onChange callback via context.

#### Props
- `children`: `Snippet` (optional). Radio items and other content.
- `value`: `string` (required). The currently selected radio value.
- `onvaluechange`: `(value: string) => void` (optional). Callback fired when the selected radio changes.

---

### ContextMenuRadioItem.svelte

A single radio option within a `ContextMenuRadioGroup`. Shows a filled circle icon when selected.

#### Props
- `children`: `Snippet` (optional). Content of the radio item.
- `class`: `string` (default: `''`). Additional CSS classes.
- `disabled`: `boolean` (default: `false`). Disables the item.
- `value`: `string` (required). The value this radio item represents.
- `...rest`: `HTMLButtonAttributes` — all standard HTML button attributes are spread onto the element.

---

### ContextMenuSeparator.svelte

A visual horizontal divider between menu sections.

#### Props
- `class`: `string` (default: `''`). Additional CSS classes.
- `...rest`: `HTMLAttributes<HTMLDivElement>` — all standard HTML div attributes are spread onto the element.

---

### ContextMenuLabel.svelte

A non-interactive text label for visually grouping menu items.

#### Props
- `children`: `Snippet` (optional). The label text.
- `class`: `string` (default: `''`). Additional CSS classes.
- `...rest`: `HTMLAttributes<HTMLDivElement>` — all standard HTML div attributes are spread onto the element.

---

### ContextMenuGroup.svelte

A semantic grouping wrapper for menu items (renders with `role="group"`).

#### Props
- `children`: `Snippet` (optional). Grouped items.
- `class`: `string` (default: `''`). Additional CSS classes.
- `...rest`: `HTMLAttributes<HTMLDivElement>` — all standard HTML div attributes are spread onto the element.

---

### ContextMenuShortcut.svelte

Displays keyboard shortcut hints aligned to the right of a menu item.

#### Props
- `children`: `Snippet` (optional). Shortcut text content (e.g., "Ctrl+C").
- `class`: `string` (default: `''`). Additional CSS classes.
- `...rest`: `HTMLAttributes<HTMLSpanElement>` — all standard HTML span attributes are spread onto the element.

---

### ContextMenuSub.svelte

Root container for a submenu. Manages the submenu's open/close state and a delayed close timer for mouse-leave behavior.

#### Props
- `children`: `Snippet` (optional). The submenu's trigger and content.

---

### ContextMenuSubTrigger.svelte

The trigger item that opens a submenu on hover, click, or ArrowRight key. Displays a chevron icon.

#### Props
- `children`: `Snippet` (optional). Content of the submenu trigger.
- `class`: `string` (default: `''`). Additional CSS classes.
- `disabled`: `boolean` (default: `false`). Disables the submenu trigger.
- `...rest`: `HTMLButtonAttributes` — all standard HTML button attributes are spread onto the element.

---

### ContextMenuSubContent.svelte

The floating panel containing submenu items. Positions itself relative to the submenu trigger.

#### Props
- `children`: `Snippet` (optional). Submenu items and content.
- `class`: `string` (default: `''`). Additional CSS classes.
- `...rest`: `HTMLAttributes<HTMLDivElement>` — all standard HTML div attributes are spread onto the element.

---

## Design System Tokens & Styling

JamUI uses Tailwind CSS utility classes and custom theme tokens:
- **Background**: `#1713c8` (`bg-jam-bg`)
- **Surface**: `#1410bd` (`bg-jam-surface`)
- **Input Surface**: `#0f0ca4` (`bg-jam-input`)
- **Primary Text**: `#fbfbff` (`text-jam-text`)
- **Muted Text**: `#d8d7ff` (`text-jam-muted`)
- **Mono Font**: `font-jam-mono`
- **Hard Shadow**: `shadow-jam-hard-sm`, `shadow-jam-hard-md`

#### Variants Style mapping:
- **ContextMenuContent panel**: White background (`bg-jam-text`), border `border-jam-bg/15`, hard blue shadow (`shadow-jam-hard-blue-sm`), minimum width `min-w-[12rem]`, z-index `z-[9999]`. Animates from `opacity-0 scale-95` to `opacity-100 scale-100` over 150ms.
- **ContextMenuSubContent panel**: Same as Content but z-index `z-[10000]` (higher to stack above parent menus).
- **ContextMenuItem (enabled, normal)**: `text-jam-bg hover:bg-jam-bg hover:text-jam-text focus:bg-jam-bg focus:text-jam-text`.
- **ContextMenuItem (enabled, destructive)**: `text-jam-danger hover:bg-jam-danger hover:text-jam-text focus:bg-jam-danger focus:text-jam-text`.
- **ContextMenuItem (disabled)**: `pointer-events-none cursor-not-allowed text-jam-bg/35`.
- **ContextMenuCheckboxItem**: `text-jam-bg hover:bg-jam-bg hover:text-jam-text focus:bg-jam-bg focus:text-jam-text`.
- **ContextMenuRadioItem**: Same styling as CheckboxItem.
- **ContextMenuSubTrigger (enabled)**: Same as ContextMenuItem normal.
- **ContextMenuSubTrigger (disabled)**: Same as ContextMenuItem disabled.
- **ContextMenuSeparator**: `my-1 h-px bg-jam-bg/10`.
- **ContextMenuLabel**: `px-3 py-1.5 text-[10px] font-black uppercase tracking-[0.18em] text-jam-bg/50`.
- **ContextMenuShortcut**: `ml-auto pl-4 text-[10px] font-medium tracking-[0.1em] text-jam-bg/40`.
- **All menu items**: `px-3 py-2 text-xs font-bold font-jam-mono tracking-[0.06em]` with `transition-colors duration-100`.

## Examples

### Basic Context Menu
```svelte
<script lang="ts">
  import {
    ContextMenu,
    ContextMenuTrigger,
    ContextMenuContent,
    ContextMenuItem,
    ContextMenuSeparator,
    ContextMenuLabel
  } from 'jamui';
</script>

<ContextMenu>
  <ContextMenuTrigger class="flex h-40 w-full items-center justify-center border border-dashed border-jam-text/30 text-jam-text">
    Right-click here
  </ContextMenuTrigger>
  <ContextMenuContent>
    <ContextMenuLabel>Actions</ContextMenuLabel>
    <ContextMenuItem onclick={() => console.log('Copy')}>Copy</ContextMenuItem>
    <ContextMenuItem onclick={() => console.log('Paste')}>Paste</ContextMenuItem>
    <ContextMenuSeparator />
    <ContextMenuItem destructive onclick={() => console.log('Delete')}>Delete</ContextMenuItem>
  </ContextMenuContent>
</ContextMenu>
```

### Context Menu with Checkboxes and Radio Group
```svelte
<script lang="ts">
  import {
    ContextMenu,
    ContextMenuTrigger,
    ContextMenuContent,
    ContextMenuItem,
    ContextMenuCheckboxItem,
    ContextMenuRadioGroup,
    ContextMenuRadioItem,
    ContextMenuSeparator,
    ContextMenuLabel,
    ContextMenuShortcut
  } from 'jamui';

  let showLineNumbers = $state(true);
  let wordWrap = $state(false);
  let theme = $state('dark');
</script>

<ContextMenu>
  <ContextMenuTrigger class="flex h-40 w-full items-center justify-center border border-dashed border-jam-text/30 text-jam-text">
    Right-click for options
  </ContextMenuTrigger>
  <ContextMenuContent>
    <ContextMenuLabel>View</ContextMenuLabel>
    <ContextMenuCheckboxItem checked={showLineNumbers} oncheckedchange={(v) => showLineNumbers = v}>
      Line Numbers
    </ContextMenuCheckboxItem>
    <ContextMenuCheckboxItem checked={wordWrap} oncheckedchange={(v) => wordWrap = v}>
      Word Wrap
    </ContextMenuCheckboxItem>
    <ContextMenuSeparator />
    <ContextMenuLabel>Theme</ContextMenuLabel>
    <ContextMenuRadioGroup value={theme} onvaluechange={(v) => theme = v}>
      <ContextMenuRadioItem value="dark">Dark</ContextMenuRadioItem>
      <ContextMenuRadioItem value="light">Light</ContextMenuRadioItem>
      <ContextMenuRadioItem value="system">System</ContextMenuRadioItem>
    </ContextMenuRadioGroup>
    <ContextMenuSeparator />
    <ContextMenuItem>
      Settings
      <ContextMenuShortcut>Ctrl+,</ContextMenuShortcut>
    </ContextMenuItem>
  </ContextMenuContent>
</ContextMenu>
```

### Context Menu with Submenu
```svelte
<script lang="ts">
  import {
    ContextMenu,
    ContextMenuTrigger,
    ContextMenuContent,
    ContextMenuItem,
    ContextMenuSub,
    ContextMenuSubTrigger,
    ContextMenuSubContent,
    ContextMenuSeparator
  } from 'jamui';
</script>

<ContextMenu>
  <ContextMenuTrigger class="flex h-40 w-full items-center justify-center border border-dashed border-jam-text/30 text-jam-text">
    Right-click with submenu
  </ContextMenuTrigger>
  <ContextMenuContent>
    <ContextMenuItem>New File</ContextMenuItem>
    <ContextMenuItem>Open</ContextMenuItem>
    <ContextMenuSeparator />
    <ContextMenuSub>
      <ContextMenuSubTrigger>Export As</ContextMenuSubTrigger>
      <ContextMenuSubContent>
        <ContextMenuItem>PDF</ContextMenuItem>
        <ContextMenuItem>PNG</ContextMenuItem>
        <ContextMenuItem>SVG</ContextMenuItem>
      </ContextMenuSubContent>
    </ContextMenuSub>
    <ContextMenuSeparator />
    <ContextMenuItem destructive>Exit</ContextMenuItem>
  </ContextMenuContent>
</ContextMenu>
```

## Accessibility & Interactions

- **Keyboard Navigation**:
  - `ContextMenu key (Shift+F10)`: Opens the context menu from the trigger element.
  - `ArrowDown`: Moves focus to the next enabled menu item.
  - `ArrowUp`: Moves focus to the previous enabled menu item.
  - `Home`: Moves focus to the first enabled menu item.
  - `End`: Moves focus to the last enabled menu item.
  - `Enter` / `Space`: Activates the focused menu item.
  - `Escape`: Closes the entire context menu.
  - `ArrowRight` (on SubTrigger): Opens the submenu.
  - `ArrowLeft` (on SubTrigger): Closes the submenu.
  - `ArrowLeft` (in SubContent): Closes the submenu and returns focus to the trigger.
- **ARIA attributes managed**:
  - `role="button"` and `aria-haspopup="menu"` on the trigger.
  - `aria-expanded` on the trigger indicates menu open state.
  - `role="menu"` on Content and SubContent panels.
  - `role="menuitem"` on ContextMenuItem and ContextMenuSubTrigger.
  - `role="menuitemcheckbox"` on ContextMenuCheckboxItem with `aria-checked`.
  - `role="menuitemradio"` on ContextMenuRadioItem with `aria-checked`.
  - `role="separator"` with `aria-orientation="horizontal"` on ContextMenuSeparator.
  - `role="group"` on ContextMenuGroup.
  - `aria-haspopup="menu"` and `aria-expanded` on ContextMenuSubTrigger.
  - `data-context-menu-layer` on menu panels for click-outside detection.
