# JamUI — RadioGroup

> A monospace, high-contrast radio button group with vertical and horizontal orientations.

## RadioGroup API & Usage Guide

The JamUI RadioGroup is a set of Svelte 5 components that build an accessible radio button group. It uses Svelte 5 Runes (`$state`, `$derived`, `$props`, `$bindable`) and context-based state management. The group supports vertical and horizontal layouts, group-level and per-item disabled states, and full WAI-ARIA radio group compliance.

### Component Structure

- `RadioGroup`: Root container that manages the selected value, group name, and orientation. Provides context to child `RadioItem` components.
- `RadioItem`: Individual radio button option rendered as a styled label with a hidden native `<input type="radio">`.

---

## Components API

### RadioGroup.svelte

Root wrapper component that manages the group state and provides context.

#### Props
- `children`: `Snippet` (optional). Content rendered inside the radio group (typically `RadioItem` components).
- `class`: `string` (default: `''`). Additional CSS classes for the root container div.
- `disabled`: `boolean` (default: `false`). Disables all radio items in the group.
- `label`: `string` (default: `''`). Accessible label for the radio group, rendered as `aria-label`.
- `name`: `string` (required). The `name` attribute shared by all radio inputs in the group.
- `orientation`: `'horizontal' | 'vertical'` (default: `'vertical'`). Layout direction. Horizontal uses `flex flex-wrap gap-2`; vertical uses `grid gap-2`.
- `value`: `string` (bindable, default: `''`). The currently selected value.
- Inherits all standard HTML `<div>` attributes (except `class` and `role`).

---

### RadioItem.svelte

Individual radio button option. Must be a child of `RadioGroup`.

#### Props
- `class`: `string` (default: `''`). Additional CSS classes for the label element.
- `disabled`: `boolean` (default: `false`). Disables this specific radio item. Overrides group disabled state.
- `label`: `string` (required). Display text for the radio option.
- `value`: `string` (required). The value this radio item represents.
- Inherits all standard HTML `<label>` attributes (except `class` and `for`).

## Design System Tokens & Styling

JamUI uses Tailwind CSS utility classes and custom theme tokens:
- **Background**: `#1713c8` (`bg-jam-bg`)
- **Surface**: `#1410bd` (`bg-jam-surface`)
- **Input Surface**: `#0f0ca4` (`bg-jam-input`)
- **Primary Text**: `#fbfbff` (`text-jam-text`)
- **Muted Text**: `#d8d7ff` (`text-jam-muted`)
- **Mono Font**: `font-jam-mono`
- **Hard Shadow**: `shadow-jam-hard-sm`, `shadow-jam-hard-md`

#### RadioItem state classes:
- **Checked**: `border-jam-text bg-jam-text text-jam-bg shadow-jam-hard-md` — solid white background with hard shadow.
- **Unchecked**: `border-jam-text/35 bg-transparent text-jam-muted hover:border-jam-text hover:bg-jam-text/10 hover:text-jam-text` — transparent with muted text and hover highlight.
- **Disabled**: `border-jam-text/20 bg-jam-surface/30 text-jam-text/30 cursor-not-allowed` — faded, non-interactive.
- **Base (all states)**: `flex min-h-11 cursor-pointer select-none items-center justify-center gap-2 rounded-none border px-4 text-xs font-black uppercase tracking-[0.18em] transition-all duration-150 ease-out focus-within:ring-2 focus-within:ring-jam-text focus-within:ring-offset-2 focus-within:ring-offset-jam-bg`.

## Examples

### Basic Vertical RadioGroup
```svelte
<script lang="ts">
  import { RadioGroup, RadioItem } from 'jamui';

  let selected = $state('option-1');
</script>

<RadioGroup name="choices" label="Choose an option" bind:value={selected}>
  <RadioItem value="option-1" label="Option 1" />
  <RadioItem value="option-2" label="Option 2" />
  <RadioItem value="option-3" label="Option 3" />
</RadioGroup>

<p>Selected: {selected}</p>
```

### Horizontal RadioGroup with Disabled Item
```svelte
<script lang="ts">
  import { RadioGroup, RadioItem } from 'jamui';

  let selected = $state('small');
</script>

<RadioGroup
  name="size"
  label="Select size"
  orientation="horizontal"
  bind:value={selected}
>
  <RadioItem value="small" label="Small" />
  <RadioItem value="medium" label="Medium" />
  <RadioItem value="large" label="Large" disabled />
</RadioGroup>
```

### Group Disabled
```svelte
<script lang="ts">
  import { RadioGroup, RadioItem } from 'jamui';

  let selected = $state('a');
</script>

<RadioGroup name="disabled-group" label="Disabled group" disabled bind:value={selected}>
  <RadioItem value="a" label="Alpha" />
  <RadioItem value="b" label="Beta" />
  <RadioItem value="c" label="Gamma" />
</RadioGroup>
```

## Accessibility & Interactions

- **Keyboard Navigation**:
  - The group uses standard native radio input behavior. Arrow keys navigate between radio items within the group (managed by the browser via the shared `name` attribute).
  - `Tab` moves focus into/out of the group.
- **ARIA attributes managed**:
  - `role="radiogroup"` on the root container.
  - `aria-label` set to the `label` prop on the root container.
  - Native `<input type="radio">` elements are visually hidden (`sr-only`) and handle checked state, `name`, `value`, and `disabled` natively.
  - The `<label>` element wraps the input, providing implicit accessibility.
